# Usage

Just copy the folder tiny-clos to the "~~lib" directory of your Gambit 
installation.

Usage:

```scheme
(##include "~~lib/tiny-clos/tiny-clos.scm")
```

or

```scheme
(##include "~~lib/tiny-clos/core.scm")
```
